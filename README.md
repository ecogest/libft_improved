# libft_improved

## Description

My libft + gnl + ft_printf library. Gathered from my previous projects.

## Build

Creates `libft.a`.

### Classic build

`make`

### Quick build with a simple main

`./run.sh`

- standard libft
- gnl
- ft_printf

- libftprintf.h
- get_next_line.h
